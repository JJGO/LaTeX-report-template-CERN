LaTeX-report-template-CERN
==========================

Template in LaTeX for openlab Summer Student report at CERN  
Due to font selections, please compile with `lualatex` i.e.

	lualatex CoSSRT.tex

If you are using BibTeX please follow:

	lualatex CoSSRT.tex
	bibtex CoSSRT
	lualatex CoSSRT.tex
	lualatex CoSSRT.tex
